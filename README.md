# Multi-Tenant SaaS Platform

This project is a basic implementation of a multi-tenant SaaS backend system using Docker, Node.js, Express, and PostgreSQL.

## Features Implemented
- Multi-tenant database structure
- Backend API with health check
- PostgreSQL database with schema and seed data
- Dockerized backend and database
- One-command setup using Docker Compose

## Tech Stack
- Node.js
- Express.js
- PostgreSQL
- Docker & Docker Compose
